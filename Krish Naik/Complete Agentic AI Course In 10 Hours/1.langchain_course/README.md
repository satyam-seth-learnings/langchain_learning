- Initialize Project 

```sh
uv init
```

- Create Virtual ENV

```sh
uv venv
```

- Activate Virtual ENV

```sh
source .venv/bin/activate
```

- Install Dependencies

```sh
uv add -r requirements.txt
```

- Get [GRAQ API Key](https://console.groq.com/keys)
